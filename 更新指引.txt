文案小程序 更新指引

一、首次使用
1. 下载最新版压缩包：
   https://github.com/canglang-88/wenan-app/releases/latest/download/wenan_app_update.zip
2. 解压压缩包。
3. 保持整个 wenan_app 文件夹完整，不要只单独拿 app.py。
4. 推荐放到 D 盘，例如：D:\wenan_app。
5. 首次打开会显示本机设备码。
6. 把设备码发给管理员获取授权秘钥。
7. 输入授权秘钥后即可使用。
8. 需要桌面图标时，双击“创建桌面快捷方式.bat”，会自动在桌面生成“文案小程序”。
9. 进入程序后，也可点击底部“建快捷方式”，一键创建桌面快捷方式。
10. 也可以双击“文案中枢.exe”或“启动文案小程序.bat”打开程序。

二、更新程序
1. 先关闭正在运行的“文案中枢”程序。
2. 如果本地有自己额外修改过的文案，可以先备份旧程序里的 data 文件夹。
   data 文件夹保存全部文案数据。
3. 用新版程序文件覆盖旧版程序文件。
4. 如果要同步云端最新文案库，请允许新版 data 文件夹覆盖旧版 data 文件夹。
5. 重新打开桌面快捷方式。
6. 已经激活过的电脑更新后不会重复要求授权，除非换电脑、重装系统、删除本地授权文件，或授权秘钥到期。
7. 如果自动更新窗口长时间停住，可以关闭程序后重新打开；同版本不会重复下载，只有云端版本高于本地版本才会再次提示。
8. 管理员本机的“秘钥生成器”不会随程序更新包上传，也不会被自动更新删除。
9. v1.6.7 起支持云端授权管控；已激活用户更新后不需要重新授权，但启动时会检查 GitHub 上的授权状态。

三、迁移到另一台电脑
1. 把整个 wenan_app 文件夹复制到新电脑的 D 盘。
2. 如果快捷方式打不开，请先双击程序目录里的“文案中枢.exe”。
3. 进入程序后点击底部“建快捷方式”，重新生成桌面快捷方式。
4. 正常发布版不需要安装 Python；只有源码版才需要 Python。

四、文案数据说明
1. 所有文案保存在程序内部 data 文件夹中。
2. 移动桌面文件夹不会影响程序数据。
3. GitHub 版本会同步 data 文案库，别人更新后可以看到你发布的最新文案。

五、导入和导出
1. 导入 TXT 时，程序会先显示预览，确认后才会写入。
2. 导出 TXT 时，可以选择保存位置。
3. 如果已选中文案，只导出选中的文案。
4. 如果没有选中文案，则导出当前筛选结果里的全部文案。
