# 文案中枢

本项目是一个 Windows 桌面文案管理工具，支持分类、标签、搜索、预览、复制、导入 TXT、导出 TXT、编辑、删除和自动更新。

## 使用方式

1. 安装 Python 3.12 或更高版本。
2. 下载并解压程序包。
3. 双击 `启动文案小程序.bat` 或运行：

```bat
pythonw app.py
```

## 数据位置

文案数据保存在程序目录下的 `data` 文件夹中。

GitHub 仓库会同步 `data` 文件夹。发布新版后，其他用户更新程序时也会同步获得最新文案库。

注意：如果用户在本地自行修改文案，执行更新时会以 GitHub 最新发布包中的 `data` 为准。

## 自动更新

程序启动后会自动检查 GitHub Releases。

当发现 GitHub 上有新版时：

1. 弹出更新提示。
2. 用户点击确认。
3. 自动下载更新包。
4. 显示下载进度条。
5. 下载完成后自动覆盖程序文件和 `data` 文案库。
6. 自动重启程序。
