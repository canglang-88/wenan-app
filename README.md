# 文案中枢

本项目是一个 Windows 桌面文案管理工具，支持分类、标签、搜索、预览、复制、导入 TXT、导出 TXT、编辑、删除和自动更新。

## 使用方式

1. 安装 Python 3.12 或更高版本。
2. 下载最新版压缩包：

```text
https://github.com/canglang-88/wenan-app/releases/latest/download/wenan_app_update.zip
```

3. 解压程序包。
4. 首次启动会显示本机设备码，请把设备码发给管理员获取授权秘钥。
5. 输入授权秘钥激活后即可使用。
6. 进入程序后可点击底部 `建快捷方式`，一键创建桌面快捷方式。
7. 也可以双击 `启动文案小程序.bat` 或运行：

```bat
pythonw app.py
```

## 数据位置

文案数据保存在程序目录下的 `data` 文件夹中。

GitHub 仓库会同步 `data` 文件夹。发布新版后，其他用户更新程序时也会同步获得最新文案库。

注意：如果用户在本地自行修改文案，执行更新时会以 GitHub 最新发布包中的 `data` 为准。

## 自动更新

程序启动后会自动检查 GitHub Releases。

当发现 GitHub 上有新版时：

1. 弹出更新提示。
2. 用户点击确认。
3. 自动下载更新包。
4. 显示下载进度条。
5. 下载完成后自动覆盖程序文件和 `data` 文案库。
6. 自动重启程序。

已激活的电脑更新后不会重复要求授权，除非更换电脑、重装系统、删除本地授权文件，或授权秘钥到期。

如果更新窗口长时间停住，可以关闭程序后重新打开；程序只会在云端版本高于本地版本时再次提示更新，同版本不会反复下载。
